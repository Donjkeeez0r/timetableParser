# 📅 timetableParser

Скрипт для парсинга расписания занятий с университетского портала. Авторизуется через Puppeteer, получает куки и парсит расписание нужной группы с помощью Cheerio.

---

## ⚙️ Как это работает

1. Puppeteer открывает браузер и авторизуется на портале
2. После входа забираются куки сессии
3. С помощью Cheerio парсится страница расписания нужной группы
4. Выводится готовое расписание в читаемом формате

---

## 🚀 Установка

```bash
git clone https://github.com/Donjkeeez0r/timetableParser.git
cd timetableParser
npm install
```

Создай файл `.env` в корне проекта:

```env
MY_LOGIN=твой_логин
MY_PASSWORD=твой_пароль

LOGIN_URL=https://...       # страница входа
FACUL1=https://...            # ссылка на расписание факультета 1
FACUL2=https://...            # ссылка на расписание факультета 2

TIMETABLE_URL=https://...   # страница с расписанием
MY_PASSWORD=твой_пароль
DEFAULT_GROUP=ГруппаXX
DEFAULT_FACULTY=''
```

---

## ▶️ Запуск

```bash
node index.js [группа] [номер_недели] [факультет]
```

**Примеры:**

```bash
# Используя значения из .env
node index.js

# Конкретная группа, первая неделя, факультет ИАТУ
node index.js ПИН-21 0 ИАТУ
```

| Аргумент | Описание | По умолчанию |
|---|---|---|
| `группа` | Название группы | `DEFAULT_GROUP` из `.env` |
| `номер_недели` | Индекс недели (0 = первая) | `0` |
| `факультет` | ФИСТ или ИАТУ | `DEFAULT_FACULTY` из `.env` |
